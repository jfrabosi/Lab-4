Detailed Description

Activates a step response for an RC circuit and plots the data and time constant.

Sends characters to a Nucleo set up for an RC step response test, then reads the data and plots it. Additionally, calculates the time constant of the circuit according to the rule that in 1 second, the response from an RC circuit will have 36.8% voltage remaining from its initial charge (aka, 1/e).
