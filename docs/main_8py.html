Detailed Description

Initiates a step response and sends data to MCU over USB VCP.

After recieving a character via USB VCP from an MCU (Nucleo), initiates a step response on an RC circuit as shown in Lab4 instructions. Then, sends the data over USB VCP to the MCU for plotting and data handling.
